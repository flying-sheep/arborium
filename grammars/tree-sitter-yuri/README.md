# tree-sitter-yuri

Yuri grammar for [tree-sitter](https://github.com/tree-sitter/tree-sitter). Incomplete and not up-to-date with newer changes to the language.

NOTE: Yuri is in early stages of development.
This repository does not generate bindings for every language
that tree-sitter provides, only for the ones we know
will be used early in its lifetime.
