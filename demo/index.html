What do we want? Syntax highlighting.

When do we want it? Now. But also good.

Arborium is a curated collection of tree-sitter grammars for Rust — server-side or client-side via WebAssembly.

The problem

For every language you want to highlight, you have to figure out:

  • Which grammar is currently maintained?
  • Is it up to date with tree-sitter?
  • Does the generated C code actually compile?
  • Does it compile to WASM without timing out?
  • Do you need to provide C library functions yourself?
  • What's the license?
  • Does it have highlight queries? Do they work?

Multiply by 80 languages. Have fun.

The solution

Arborium handles all of this. Every grammar is tested, patched, and ships with working highlight queries. If it's in the collection, it works — native and WASM.
